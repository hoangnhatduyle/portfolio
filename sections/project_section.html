Huffman Coding CompressionHuffman coding (also known as Huffman Encoding) is an algorithm for doing data compression, and it forms the basic idea behind file compression. The idea is to use “variable-length encoding”. It is due to the fact that some characters occur more frequently than others in a text to design an algorithm that can represent the same piece of text using a lesser number of bits. We can then reference to the Huffman tree to decode any given string of characters.

Radix Project Radix project is written in Intel 80x86 Assembly Language. The program will first take in radices for input, perform basic operations and output the results in radix from 2 (binary) to 62. Once it is complete, it will prompt the user if they wish to repeat. In addition, none of the Irvine32 functions are to be used, since all the procedures were manually rewritten (reading the input/converting it to a usable value).

Huffman Coding CompressionHuffman coding (also known as Huffman Encoding) is an algorithm for doing data compression, and it forms the basic idea behind file compression. The idea is to use “variable-length encoding”. It is due to the fact that some characters occur more frequently than others in a text to design an algorithm that can represent the same piece of text using a lesser number of bits. We can then reference to the Huffman tree to decode any given string of characters.

Sesquicentennial Bone Conduction RailingWith the sponsorship of the College of Arts and Letters & UT green fund, we were tasked with adding a unique art installation on campus that students and guests would be able to interact with that shared the history of Toledo. The design was a prototype fixture that can transmit sound through the bones in the user's forearms to allow them to hear the sounds of campus, similar to the art installation in Dresden. Our railing has the ability to be installed in various locations across campus, including Carlson Library, the UT Health Science campus, and Engineering Campus.
